# Differ

Differ is a simple plugin allowing you to perform git-like base-head comparisons directly inside studio.

## Experimental

This plugin is currently unfinished and still has a long way to go. Planned features are as follows:
- Tabs: would allow you to seamlessly switch between switching the instance you're viewing
- Property Diffs: support for showing differences in properties
- Model Visualizer: differences to models would be displayed in a `ViewportFrame` for a quick and easy view

While these features are planned, I do not promise an addition of all of them as this plugin was made for personal use - specifically for tracking script changes.

## Terms of Use

- Any modifications made to Differ must be shared publicly with full source code, so others can benefit from your changes.
